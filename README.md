# Project `Pokedex 포켓몬도감` 📚📖

## 🖥 Project Summary

- 포켓몬 도감 사전 프로젝트

## 👩🏻‍💻 Front-end Work Details

###

- `Pokemons List`: 포켓몬 이름 리스트
  - 무한 스크롤 pagination
- `Detail`: 포켓몬 디테일 정보 페이지
  - name,abilities,weigth,height 불러오기
  - evolution 진화 단계 트리 생성 및 노출
- `SearchBar`: 포켓몬 번호로 서치
  - 번호 검색 기능
  - 번호 reset 버튼
